<details>
<summary><b>⚙️ One-time setup to activate this animation (click to expand)</b></summary>

<br/>

This image stays blank until you add a small GitHub Action that generates it. Steps:

1. In your `Srilakshmi-Dirisala/Srilakshmi-Dirisala` repo, create the folder path `.github/workflows/`.
2. Add a file there named `snake.yml` with this content:

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 0 * * *"   # runs once a day
  workflow_dispatch:        # lets you trigger it manually
  push:
    branches:
      - main

jobs:
  generate:
    runs-on: ubuntu-latest
    permissions:
      contents: write
    steps:
      - name: Generate the snake SVG
        uses: Platane/snk@v3
        with:
          github_user_name: Srilakshmi-Dirisala
          outputs: dist/github-contribution-grid-snake.svg

      - name: Push snake to the output branch
        uses: crazy-max/ghaction-github-pages@v4
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

3. Go to **Settings → Actions → General → Workflow permissions** in your repo and select **"Read and write permissions"**, then save. (Without this, the Action can't push the generated image.)
4. Commit and push. Go to the **Actions** tab and confirm the "Generate Snake Animation" workflow runs successfully — it creates an `output` branch with the SVG.
5. That's it — the image above will now render automatically, and it refreshes daily to reflect your latest contributions.
